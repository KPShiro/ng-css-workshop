How to change Flexbox direction?

If you need to change the layout, for example render elements from top to bottom, you will need to use flex-direction property. This will switch the main axis between horizontal and vertical. There are a few options available:

How to align items on the main axis?

While working with the Flexbox you will need to somehow align items within your container. It is also a very easy task that will require from you only the knowledge of justify-content property. Here are the possible values:
