CSS Flexbox

During the CSS stone age, developers were creating layouts with float and position until one day Flexbox will be introduced and the world will never be the same.

What is Flexbox?

Flexbox is a CSS module. It involves a lot of things including its whole set of properties. Some of them are meant to be set on the container (parent element, known as “flex container”) whereas the others are meant to be set on the children (“flex items”).

How to start using Flexbox?

  • Create a structure of a container and it's children in HTML.
  • In the CSS file, add display: flex to the container element.

    This will add 2 invisible axis (main axis and cross axis) to the element. Our items will no longer be stacked one above the other, but placed along the main axis. By default it is horizontal, but it can be changed with flex-direction property.
