CSS Flexbox - Items

Flexbox gives you also control over the items inside flex containers. All properties below can be used only on the flex items.

flex-grow

When styling flexbox items, it's common to specify their growth. You can do that by controlling unitless property called flex-grow. The value serves as the proportion and it allows the item to grow if there is a space to do so. In the example below, second item fills the space.

flex-shrink

Another usefull property - flex-shrink - allows to specify how fast certain item can shrink. This property also takes unitless value, same as flex-grow. The higher number set, the faster your item will shrink. If you want to disable shrinking completely, set a value to 0.

flex-basis

This property defines the size of the item, before the remaining space is distributed. If your item has already some width (e.g. 128px), you can override that with flex-basis.

Usefull shorthand - flex

Personally I almost never use the properties above separately. Most of the time I use a shorthand - flex. This property takes 3 values: flex-grow, flex-shrink and flex-basis - last two are optional.

Override item's position

There might be a situation where you will have to positon a certain flex item differently from the rest. You can do that with the align-self property. This will give you control over the item's position along the cross axis.

Flex item order

There is one more flex item property - order. It lets you change the order of the items in the container. Remember to use it only if you absolutely have to, because it messes with the semantics and the accessibility of your HTML.

All items have the order property value set to 0 by default. In the example below, I set the order: -1; for the last item, so it can be displayed as the first one in the row.
